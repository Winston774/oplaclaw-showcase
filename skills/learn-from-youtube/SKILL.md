# Learn From YouTube

你是一位 AI 代理助手，具備「從 YouTube 影片學習並協助執行任務」的能力。

---

## 觸發條件

當使用者訊息中包含 YouTube 影片連結（`youtube.com/watch` 或 `youtu.be/`）時：

- **有學習意圖**（例如：學習、學、了解、幫我、看這個、教我、試試看、怎麼做）→ **直接執行本技能**
- **意圖不明確**（只貼 URL，沒有其他說明）→ 主動詢問：「你想從這支影片學習並執行相關任務嗎？」
- **只是討論或分享影片**（例如「你看過這個嗎？」）→ **不觸發本技能**，正常回應即可

---

## 執行步驟

### 第一步：確認字幕工具是否就緒

嘗試使用 `youtube-transcript-yt-dlp` 技能取得字幕。

**若技能尚未安裝**，向使用者說明：

> 需要先安裝字幕擷取技能，請在終端機執行：
>
> ```
> npx clawhub@latest install itzsubhadip/youtube-transcript-yt-dlp
> ```
>
> 安裝完成後，重新貼上影片連結即可繼續。

**若技能已就緒**，直接進行下一步，不顯示任何提示。

---

### 第二步：取得字幕 → 生成摘要 → 分析

1. 優先取得**英文字幕**（若無英文字幕，改取任何可用語言）
2. 將字幕內容翻譯成**繁體中文**（**不顯示逐字稿**）
3. 根據字幕全文，生成一份 **500–1000 字的繁體中文影片摘要**，描述影片的主要內容、流程與重點概念
4. 進行以下四項分析：

**A. 影片核心能力**
條列 3–5 項「學完後你將能夠⋯⋯」的具體能力

**B. 所需環境與工具清單**
列出影片提到的所有工具、平台、API、套件及版本需求

**C. 環境差距分析**
主動掃描使用者目前的環境（已安裝工具、現有設定），與 B 項清單逐一對照：
- ✅ 已就緒的項目
- ❌ 缺少或需要設定的項目

**D. 建議執行順序**
根據差距分析，列出最短路徑的設定步驟

---

### 第三步：提案並等待確認

完成分析後，向使用者呈現以下內容，然後**停止並等待使用者回應**：

```
我已讀完這支影片的完整內容。

📝 影片摘要：
[500–1000 字的繁體中文摘要]

🎯 學完後你將具備的能力：
[A 項清單]

🔍 環境差距分析：
[C 項清單]

📋 建議設定步驟：
[D 項清單]

請問要開始嗎？或者告訴我你想從哪個步驟切入。
```

---

### 第四步：建立環境並執行任務

使用者確認後，依步驟協助建立環境並執行任務。

- 遇到需要使用者手動操作的步驟時，**暫停說明並等待確認**後再繼續
- 每個步驟完成後確認結果，再進行下一步
